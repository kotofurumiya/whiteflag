# Whiteflag

## Windows/macOSからの利用

[https://sbfl.net/app/whiteflag](https://sbfl.net/app/whiteflag)
からダウンロードしてインストールしてください。

## ソースからビルドする

依存パッケージをインストールします。

```
$ npm install
```

ビルドします。

```
$ npm run build
```

インストーラを作成します。

```
$ npm run package
```

Win/macそれぞれのインストーラのみを作成することもできます。

```
$ npm run package:win
$ npm run package:mac
```
